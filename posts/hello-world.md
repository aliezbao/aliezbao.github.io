# Hello World

欢迎来到我的博客！这是我的第一篇文章。

## 关于这个博客

这是一个使用纯 HTML、CSS 和 JavaScript 构建的静态博客网站，采用了简洁的设计风格，使用了kiro vibe coding 进行创建。

### 特点

- 简洁优雅的设计
- 前后端分离架构
- 使用 Markdown 编写文章
- 易于部署到 GitHub Pages

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- Marked.js (Markdown 解析)

## 开始写作

只需在 `posts` 目录下创建新的 Markdown 文件，然后在 `posts/index.json` 中添加文章信息即可。

```javascript
{
    "id": "your-post-id",
    "title": "文章标题",
    "date": "2026-02-12",
    "file": "your-post.md",
    "excerpt": "文章摘要"
}
```

祝你写作愉快！
